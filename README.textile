h1. Alte Version

Gemeinschaft in der Version 3.1 ist eine veraltete Software mit einem Asterisk-Kern. Wenn Sie möchten, können Sie die Software natürlich weiterhin einsetzen. Wir empfehlen Ihnen aber lieber auf eine neue Version zu gehen. Weitere Informationen dazu finden Sie auf "http://www.amooma.de/gemeinschaft":http://www.amooma.de/gemeinschaft

h1. Installationsanleitung für Gemeinschaft 3.1

Eine Installationsanleitung finden Sie unter  "http://github.com/amooma/GS3/wiki/Installation":http://github.com/amooma/GS3/wiki/Installation

h1. Support und Consulting für Gemeinschaft 3.x

Kostenpflichtig bei der "AMOOMA GmbH":http://www.amooma.de
Für 200 Euro pro Stunde (plus MwSt.) können Sie Support und Consulting für Ihre Gemeinschaft Installation bekommen. Schreiben Sie uns dazu einfach eine E-Mail an info@amooma.de
