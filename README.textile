h1. Sammlung für eine Version 3.2

Im Januar 2012 wird die Version 4.0 auf Basis von FreeSWITCH veröffentlicht.
Da es noch einige Freunde der 3.xer Version auf Asterisk Basis gibt, haben wir
eine Sammlung für eine letzte 3.2 Version mit aktuellen Features und
Provisionings gestartet. Wenn bis zum 31.12.11 genug Geld reinkommt, werden
wir noch eine Version 3.2 veröffentlichen. Sonst geht es direkt zur 4.0.

Hier geht es zur Spendensammlung für die 3.2:
http://pledgie.com/campaigns/16375
